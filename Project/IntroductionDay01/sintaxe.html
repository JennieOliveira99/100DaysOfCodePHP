Introdução ao PHP

O PHP é uma linguagem de script de servidor amplamente utilizada para desenvolvimento web. Ele é especialmente adequado para criar páginas da web dinâmicas e interativas. O PHP pode ser incorporado diretamente no HTML e é compatível com uma variedade de bancos de dados, o que o torna uma escolha popular entre os desenvolvedores web.

Com o PHP, você pode realizar uma ampla gama de tarefas, incluindo manipulação de formulários, interação com bancos de dados, gerenciamento de sessões de usuário e muito mais. Sua sintaxe é semelhante a outras linguagens de programação, como C, Java e Perl, tornando-a relativamente fácil de aprender para aqueles que já estão familiarizados com programação.

Além disso, o PHP é uma linguagem de código aberto, o que significa que é gratuito para uso e distribuição. Sua comunidade ativa oferece uma vasta quantidade de recursos, bibliotecas e frameworks para facilitar o desenvolvimento web.

Dia 1-2: Introdução ao PHP: sintaxe básica, variáveis e tipos de dados

Comando echo

Comando echo forma simples:

echo 'Hello world';

Comando echo em expressão ternária:

$some_var = true;
echo ($some_var) ? 'true' : 'false';

Imprimindo aspas:

echo "Isto é uma \"aspas duplas\" dentro de uma string.";

Imprimindo com printf():

printf("Olá, meu nome é %s e eu tenho %d anos.", $nome, $idade);
